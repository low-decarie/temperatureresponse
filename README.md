Companion to:
Low-Décarie E, Boatman TG, Bennett N, Passfield W, Gavalás-Olea A, Siegel P, Geider RJ (2017) Predictions of response to temperature are contingent on model choice and data quality. Ecology and Evolution, 1–15.
